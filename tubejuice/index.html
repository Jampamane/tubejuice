Quick Start

1. Install dependencies

cd tubejuice
uv sync
# also needs ffmpeg:
brew install ffmpeg # macOS
apt install ffmpeg # Debian/Ubuntu

2. Run the server

uv run tubejuice

3. (Optional) beets config

Edit beets_config.yaml.template to customize
music directory, paths, and plugins.

Tips

• Works with YouTube Music playlists, albums, and single videos
• FLAC gives lossless quality (larger files)
• Beets queries MusicBrainz for accurate metadata
• Music saved to ./music/ folder
• Downloads run in background — you can queue multiple
